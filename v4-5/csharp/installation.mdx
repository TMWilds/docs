</Info>

To install the C# SDK:

<Steps>
<Step>
Confirm that you meet the minimum requirements. ([Prerequisites](<./../Java/Installing Java SDK.md>))
</Step>

<Step>
Prepare your environment for Ditto. ([Setting Up Your Environment](<./../Java/Installing Java SDK.md>))
</Step>

<Step>
Set up your app permissions. ([Configuring Permissions](<./../Java/Installing Java SDK.md>))
</Step>

<Step>
Add Ditto to your app. ([Integrating Ditto](<./../Java/Installing Java SDK.md>))
</Step>
</Steps>

# Prerequisites

Following are the minimum requirements that must be met before attempting to install Ditto.

- <a href="https://learn.microsoft.com/en-us/dotnet/standard/net-standard?tabs=net-standard-2-1#net-standard-versions" target="_blank">.NET Standard 2.1</a>&#x20;
- <a href="https://dotnet.microsoft.com/en-us/download/dotnet/3.0" target="_blank">.NET Core 3</a>&#x20;
- <a href="https://dotnet.microsoft.com/en-us/download/dotnet-framework/net40" target="_blank">.NET Framework 4.0</a>

# Installing Dependencies

Using either the NuGet package manager, .NET CLI, or adding a reference to your `.csproj` XML file:

<Tabs>
<Tab title="NuGet Package Manager">
```
Install-Package Ditto -Version 4.3.0
```
</Tab>

<Tab title=".NET CLI">
```
dotnet add package Ditto --version 4.3.0
```
</Tab>

<Tab title="CSProj XML">
```
<PackageReference Include="Ditto" Version="4.3.0" />
```
</Tab>
</Tabs>

<Info>
For the official .NET standard for Ditto, see the official NuGet reference at Packages > <a href="https://www.nuget.org/packages/Ditto/" target="_blank">Ditto</a>.
</Info>

# Integrating Ditto

Integrate Ditto with your app:

<Info>
For instructions on how to obtain your access credentials, see [Onboarding](/v4-4/onboarding). For an introduction to authentication in Ditto, see *Ditto Basics* > [Authentication and Initialization](</v4-5/basics/authentication-and-initialization>).
</Info>

```csharp
try
{
    DittoLogger.SetMinimumLogLevel(DittoLogLevel.Debug);
    var ditto = new Ditto(DittoIdentity.OnlinePlayground("REPLACE_ME_WITH_YOUR_APP_ID", "REPLACE_ME_WITH_YOUR_PLAYGROUND_TOKEN", true), path);
    ditto.StartSync();
}
catch (DittoException ex)
{
    Console.WriteLine($"Ditto Error {ex.Message}");
}
```

```csharp
var docId = ditto.Store.Collection("people").Upsert(
    new Dictionary<string, object> {
    { "name", "Susan" },
    { "age", 31 },
    }
);
```
